FAQ - WebNotes+

📌 Qu'est-ce que WebNotes+ ?

WebNotes+ est une extension de navigateur permettant de prendre, gérer, organiser et résumer des notes directement depuis vos pages web visitées.

🧠 Comment fonctionne le résumé automatique ?

Le texte de la page est extrait, envoyé à un serveur hébergé sur Render.com qui utilise un modèle IA de Hugging Face pour générer un résumé. Ce résumé est renvoyé à l’extension et peut être édité avant sauvegarde.

🔐 Où sont stockées mes notes ?

Les notes sont uniquement enregistrées localement dans votre navigateur via chrome.storage.local. Aucune donnée n'est transmise à des serveurs externes.

💾 Puis-je exporter mes notes ?

Oui ! Vous pouvez exporter vos notes en formats PDF, texte brut ou Markdown, directement depuis l’interface de l’extension.

🌐 Est-ce compatible avec tous les navigateurs ?

WebNotes+ est actuellement disponible pour Google Chrome via le Chrome Web Store, firefox et OperaGX.

📥 Où puis-je télécharger l’extension ?

Depuis le Chrome Web Store : Lien de téléchargement, firefox ou OperaGX
