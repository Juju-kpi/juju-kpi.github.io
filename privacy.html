Politique de Confidentialité

Dernière mise à jour : 12 août 2025

Cette politique de confidentialité explique comment l'extension Quick Web Notes collecte, utilise et protège les informations des utilisateurs.

1. Informations collectées

Nous ne collectons ni ne stockons d’informations personnelles identifiables. Toutes les données générées par l'utilisateur (notes, titres, URLs) sont :

  • Stockées localement dans le navigateur via chrome.storage.local,
  • Non partagées avec des serveurs externes.

2. Données enregistrées localement

Les données suivantes sont enregistrées localement :

  • Titre et contenu des notes,
  • URL de la page web associée,
  • Date de création des notes.

Aucune de ces données n’est transmise à un serveur tiers.

3. Publicité

L'extension peut intégrer des publicités via des partenaires comme PropellerAds ou autres. Ces services peuvent utiliser :

  • Des cookies ou identifiants anonymes,
  • Des données non personnelles (adresse IP, navigateur, type d'appareil) pour personnaliser les publicités et mesurer leur efficacité.

Pour plus d'informations, consultez : Politique de confidentialité de PropellerAds.

4. Partage d’informations

Aucune donnée utilisateur n’est vendue, partagée ou transférée à des tiers, sauf aux fins décrites ci-dessus.

5. Sécurité

Les données sont stockées de manière sécurisée via les mécanismes de Chrome. L'extension ne transmet rien à des serveurs distants.

6. Consentement

En utilisant cette extension, vous acceptez les termes de cette politique. Si vous n’êtes pas d’accord, veuillez désinstaller l’extension.

7. Modifications

Cette politique peut être mise à jour sans préavis. La dernière version est toujours disponible sur cette page.
